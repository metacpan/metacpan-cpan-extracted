=head1 NAME

OpenAPI::Client::OpenAI::Path::skills-skill_id-content - Documentation for the /skills/{skill_id}/content path.

=head1 OPERATIONS

=head2 GET /skills/{skill_id}/content

=head3 GetSkillContent

  $client->get_skill_content({

=head4 Path/query parameters

=over

=item * C<skill_id> (in path, required, string) - The identifier of the skill to download.
