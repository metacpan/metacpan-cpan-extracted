ars_DeleteField(ctrl, schema, fieldId, deleteOption=0)

Delete the given field (specified as a field id) from the given schema on the server connected to in the given control structure. Refer to the ARS API Guide for an explanation of the deleteOption. Requires Administrator privs to use.

On success
Returns 1.
On failure
Returns 0.

Example: 

      ars_DeleteField($ctrl, "HD:HelpDesk", 536870920) || die "$ars_errstr";
ars_DeleteField was introduced in version 1.5 of ARSperl
