# NAME

Mojolicious::Plugin::Fondation::Menu - Dynamic menu management plugin for Fondation — navbar, breadcrumb

# VERSION

version 0.01

# SYNOPSIS

    # myapp.conf
    {
        'Fondation' => {
            dependencies => [
                { 'Fondation::Model::DBIx::Async' => {
                    backends => [ main => {
                        dsn          => 'dbi:SQLite:dbname=data/app.db',
                        schema_class => 'MySchema',
                    }],
                }},
                'Fondation::Menu',
            ],
        },
    }

# DESCRIPTION

[Mojolicious::Plugin::Fondation::Menu](https://metacpan.org/pod/Mojolicious%3A%3APlugin%3A%3AFondation%3A%3AMenu) provides dynamic menu management.
Menus are declared declaratively via `share/menus.json` in each plugin
and synchronized with the database via `menu sync`.

## Schema

    id          INTEGER PRIMARY KEY AUTOINCREMENT
    title       TEXT NOT NULL
    link        TEXT
    icon        TEXT          (FontAwesome class)
    icon_color  TEXT          (CSS color)
    name        TEXT NOT NULL DEFAULT 'left'  (menu group)
    condition   TEXT          ("group:NAME", "perm:NAME", or "")
    sort_order  INTEGER DEFAULT 0
    parent_id   INTEGER NOT NULL DEFAULT 0
    open_tab    INTEGER DEFAULT 0
    view_in_menu INTEGER DEFAULT 1
    description TEXT
    created_at  TEXT
    updated_at  TEXT

## menus.json format

Each plugin can ship a `share/menus.json`:

    [
      {
        "title":        "Administration",
        "link":         "/admin",
        "icon":         "fas fa-lock-open",
        "menu":         "admin_menu",
        "condition":    "group:administrateur",
        "order":        1,
        "parent_title": null,
        "description":  "Admin panel"
      },
      {
        "title":        "Users",
        "link":         "/users",
        "menu":         "admin_menu",
        "parent_title": "Administration"
      }
    ]

Fields: `title` (required), `link`, `icon`, `icon_color`,
`menu` (group name, default "left"), `condition`, `order`
(auto if absent), `parent_title`, `open_tab`, `view_in_menu`,
`description`.

## Commands

    myapp.pl menu sync       Sync menu entries from share/menus.json
