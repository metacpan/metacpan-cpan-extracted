# Contributing to Mojolicious::Plugin::Hakkefuin

Thank you for considering contributing to this project!

## How to Contribute

1. Fork the repository on GitHub  
2. Create a feature branch  
3. Make your changes, including tests  
4. Run the test suite  
5. Submit a pull request to:

https://github.com/CellBIS/mojo-hakkefuin

## Reporting Bugs

File an issue at:

https://github.com/CellBIS/mojo-hakkefuin/issues

Please include a minimal reproduction and details about your Perl version.

## Code Style

- Follow Perl Best Practices
- Keep changes minimal and performance-minded
- Use perltidy (optional but recommended)

Thank you for improving Mojolicious::Plugin::Hakkefuin!
