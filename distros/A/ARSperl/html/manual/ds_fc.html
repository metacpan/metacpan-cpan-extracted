Field Characteristics Structure

A reference to this structure is returned by the Active Link Action Structure when the action is the change in field characteristics. 
{
 "accessOption" => integer,
 "charMenu"     => string,
 "fieldId"      => integer,
 "focus"        => integer,
 "display"      => reference to Display Structure, (2.x only)
 "props"        => reference to Props Structure, (3.x only)
}
