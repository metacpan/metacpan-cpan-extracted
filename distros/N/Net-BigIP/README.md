# BigIP::REST

## DESCRIPTION

This module provides a Perl interface for communication with BigIP load-balancer
using REST interface.

## INSTALLATION

To install this module:

    $> tar xvzf BigIP-REST-X.Y.tar.gz
    $> cd BigIP-REST-X.Y
    $> perl Makefile.PL
    $> make
    $> make test
    $> make install
