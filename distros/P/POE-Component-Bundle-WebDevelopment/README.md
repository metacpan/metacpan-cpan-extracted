# NAME

POE::Component::Bundle::WebDevelopment - a collection of PoCos useful for Web Development

# CONTENTS

This distribution provides the following
[POE::Component](https://metacpan.org/pod/POE%3A%3AComponent) modules:

- [POE::Component::App::PNGCrush](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AApp%3A%3APNGCrush) - non-blocking wrapper around [App::PNGCrush](https://metacpan.org/pod/App%3A%3APNGCrush)
- [POE::Component::CSS::Minifier](https://metacpan.org/pod/POE%3A%3AComponent%3A%3ACSS%3A%3AMinifier) - non-blocking wrapper around [CSS::Minifier](https://metacpan.org/pod/CSS%3A%3AMinifier) with URI fetching abilities
- [POE::Component::JavaScript::Minifier](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AJavaScript%3A%3AMinifier) - non-blocking wrapper around [JavaScript::Minifier](https://metacpan.org/pod/JavaScript%3A%3AMinifier) with URI fetching abilities
- [POE::Component::Syntax::Highlight::CSS](https://metacpan.org/pod/POE%3A%3AComponent%3A%3ASyntax%3A%3AHighlight%3A%3ACSS) - non-blocking wrapper around [Syntax::Highlight::CSS](https://metacpan.org/pod/Syntax%3A%3AHighlight%3A%3ACSS)
- [POE::Component::Syntax::Highlight::HTML](https://metacpan.org/pod/POE%3A%3AComponent%3A%3ASyntax%3A%3AHighlight%3A%3AHTML) - non-blocking wrapper around [Syntax::Highlight::HTML](https://metacpan.org/pod/Syntax%3A%3AHighlight%3A%3AHTML)
- [POE::Component::WWW::Alexa::TrafficRank](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3AAlexa%3A%3ATrafficRank) - non-blocking wrapper around [WWW::Alexa::TrafficRank](https://metacpan.org/pod/WWW%3A%3AAlexa%3A%3ATrafficRank)
- [POE::Component::WWW::Cache::Google](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3ACache%3A%3AGoogle) - non-blocking wrapper around [WWW::Cache::Google](https://metacpan.org/pod/WWW%3A%3ACache%3A%3AGoogle)
- [POE::Component::WWW::DoctypeGrabber](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3ADoctypeGrabber) - non-blocking wrapper around [WWW::DoctypeGrabber](https://metacpan.org/pod/WWW%3A%3ADoctypeGrabber)
- [POE::Component::WWW::GetPageTitle](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3AGetPageTitle) - non-blocking wrapper around [WWW::GetPageTitle](https://metacpan.org/pod/WWW%3A%3AGetPageTitle)
- [POE::Component::WWW::HTMLTagAttributeCounter](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3AHTMLTagAttributeCounter) - non-blocking wrapper around [WWW::HTMLTagAttributeCounter](https://metacpan.org/pod/WWW%3A%3AHTMLTagAttributeCounter)
- [POE::Component::WWW::Lipsum](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3ALipsum) - non-blocking wrapper around [WWW::Lipsum](https://metacpan.org/pod/WWW%3A%3ALipsum)
- [POE::Component::WWW::WebDevout::BrowserSupportInfo](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWWW%3A%3AWebDevout%3A%3ABrowserSupportInfo) - non-blocking access to browser support API on [http://webdevout.net](http://webdevout.net)
- [POE::Component::WebService::HtmlKitCom::FavIconFromImage](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWebService%3A%3AHtmlKitCom%3A%3AFavIconFromImage) - non-blocking wrapper around [WebService::HtmlKitCom::FavIconFromImage](https://metacpan.org/pod/WebService%3A%3AHtmlKitCom%3A%3AFavIconFromImage)
- [POE::Component::WebService::Validator::CSS::W3C](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWebService%3A%3AValidator%3A%3ACSS%3A%3AW3C) - non-blocking wrapper around [WebService::Validator::HTML::W3C](https://metacpan.org/pod/WebService%3A%3AValidator%3A%3AHTML%3A%3AW3C)
- [POE::Component::WebService::Validator::HTML::W3C](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AWebService%3A%3AValidator%3A%3AHTML%3A%3AW3C) - a non-blocking POE wrapper around [WebService::Validator::HTML::W3C](https://metacpan.org/pod/WebService%3A%3AValidator%3A%3AHTML%3A%3AW3C)
