Options

 The Dynamic style-element stores information which is used during the actual presentation, and what is not related to the display of the slide itself. Actually, it is all the information shown in the control-display, if that display is enabled.

General Dynamics Options

The Dynamic style-element contains the following options:
-startPhase => number
The first phase of the slide to be shown. Usually, this will contain 0 to indicate all phases.

This option is overruled by the show's option -flushPhases.

-exportPhases => boolean
-exportPhases => [ phase number, ... ]
Usually, only the final stage of each slide is incorporated in a website or list of images, as created by an export module. When you set this on true, then all phases are included. When you assign a list, only the listed phases are included.
