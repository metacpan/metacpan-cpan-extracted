=head1 NAME

OpenAPI::Client::OpenAI::Path::organization-costs - Documentation for the /organization/costs path.

=head1 OPERATIONS

=head2 GET /organization/costs

=head3 usage-costs

  $client->usage_costs({
      body => { ... },
  });

Get costs details for the organization.

=head4 Path/query parameters

=over

=item * C<start_time> (in query, required, integer) - Start time (Unix seconds) of the query time range, inclusive.

=item * C<end_time> (in query, optional, integer) - End time (Unix seconds) of the query time range, exclusive.

=item * C<bucket_width> (in query, optional, string) - Width of each time bucket in response. Currently only C<1d> is supported,
default to C<1d> .

Allowed values: 1d

Default: 1d

=item * C<project_ids> (in query, optional, array) - Return only costs for these projects.

=item * C<api_key_ids> (in query, optional, array) - Return only costs for these API keys.

=item * C<group_by> (in query, optional, array) - Group the costs by the specified fields. Support fields include C<project_id>
, C<line_item> , C<api_key_id> and any combination of them.

=item * C<limit> (in query, optional, integer) - A limit on the number of buckets to be returned. Limit can range between 1 and
180, and the default is 7.

Default: 7

=item * C<page> (in query, optional, string) - A cursor for use in pagination. Corresponding to the C<next_page> field from
the previous response.

=back
