Geo-OGC-Service-WFS version 0.10
================================

A web feature service.

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

Geo::GDAL Geo::OGC::Service
