# Security Policy

## Reporting a Vulnerability

Please do **not** report security vulnerabilities via GitHub issues.

Instead, email **mendoza@pvv.ntnu.no** with the subject line
`[Archive::Lha] Security Vulnerability`.

Include as much detail as possible:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix

You can expect an acknowledgement within 5 business days.

## Supported Versions

Only the latest release on CPAN receives security fixes.
