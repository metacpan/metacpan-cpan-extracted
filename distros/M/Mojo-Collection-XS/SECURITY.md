# Security Policy for Mojo::Collection::XS

Thank you for taking the time to report security issues.

## Reporting a Vulnerability

If you discover a potential security vulnerability in this module, please report it privately.

Contact:

Email: yusrideb@cpan.org  
GitHub Security Advisories: https://github.com/CellBIS/mojo-collection-xs/security/advisories

We will respond as quickly as possible and coordinate a fix.
