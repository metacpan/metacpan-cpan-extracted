# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PDF::Reuse, please report it
responsibly.

**Contact:** Chris Nighswonger <cnighs@cpan.org>

Alternatively, you can open a security advisory on GitHub:
https://github.com/cnighswonger/PDF-Reuse/security/advisories/new

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 72 hours and aim to provide a fix
or mitigation plan within 30 days.
