
# NAME

Task::WebDyne::PAGI - Install PAGI dependencies for WebDyne

# SYNOPSIS

```bash
cpan Task::WebDyne::PAGI

cpanm Task::WebDyne::PAGI
```

# DESCRIPTION

Task::WebDyne::PAGI installs the Perl modules required to use PAGI with
WebDyne.
