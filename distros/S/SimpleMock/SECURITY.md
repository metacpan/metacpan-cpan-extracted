# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SimpleMock, please report it responsibly.

**Contact:** Clive Holloway <clive.holloway@gmail.com>

Please include a description of the issue, steps to reproduce, and any potential impact. You should receive a response within 72 hours.

## Scope

SimpleMock is a testing framework and is not intended for use in production environments. Security issues related to mock behavior in test suites are still welcome.
