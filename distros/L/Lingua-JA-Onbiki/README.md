# NAME

Lingua::JA::Onbiki - convert Onbiki into Boin

# SYNOPSIS

    use Lingua::JA::Onbiki qw/onbiki2boin/;

    $onbiki = 'あったか〜い';
    $boin = onbiki2boin($onbiki); # => あったかあい

# DESCRIPTION

Lingua::JA::Onbiki converts Onbiki into Boin in Japanese.
