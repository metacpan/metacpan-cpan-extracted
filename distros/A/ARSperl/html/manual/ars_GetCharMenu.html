ars_GetCharMenu(ctrl, name)

Returns a HASH consisting of menu attributes.
name is the actual menu name as shown in the ARAdmin tool.

On success
returns a HASH
On failure
returns undef

ars_GetCharMenu was introduced in version 1.3 of ARSperl.
