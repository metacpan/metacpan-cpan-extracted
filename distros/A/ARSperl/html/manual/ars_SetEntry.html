ars_SetEntry(ctrl,schema,entry_id,getTime,...)

Modifies the value of an entry in a schema.

Alternating field ids and values should be given after "getTime".

On success
Returns True (1)
On failure
Returns undefined

Example: 

      $r = ars_SetEntry($ctrl, $schema, $entryid, 0, $fid, $content);
