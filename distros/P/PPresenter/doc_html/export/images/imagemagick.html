 Portable Presenter is a package designed to give presentations. With this module, you can export your slide into a directory full of images.

Using this Module

To be able to use this exporter module, you have to add it explicitly to your show: 
   $show->addExporter('PPresenter::Export::Images::ImageMagick');

Options

The Images::ImageMagick objects have all general options from Export. Next to these you can specify:
-outputDir => directory
The directory where the images are written to. This directory is created when required. The directory is not clean-up when it already exists.
