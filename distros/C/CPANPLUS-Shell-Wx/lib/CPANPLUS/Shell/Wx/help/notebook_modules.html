wxCPAN Modules View

This is the heart of wxCPAN. It contains two panes: the Module List and the Information Panel. You can move the divider up and down to change the size of the views. The Information Pane is describe in the Information Pane section. The Module List is described here.

Populating the list

Use the Toolbar to populate the list.

Selecting a Module

Single-Clicking on a module name allows you to use the Information pane on that module. When you click on a name, it will highlight and all the items in the Information Pane will be reset.

Getting Documentation

You can double-click on a module name, and the POD Reader view will be opened with that module's (POD) Documentation. See the POD Reader section for more information.

Adding To Actions Tab

Right-click on an item and select 'Batch->[Action]' to put that action in the Actions List, for later processing.
