=head1 SYNOPSIS

Commandline:

    > pegex-forth program.forth
    > echo '3 4 + .' | pegex-forth            # Print 7

In code:

    use Pegex::Forth;
    Pegex::Forth->new->run('3 4 + .');        # Print 7
    $num = Pegex::Forth->new->run('3 4 +');   # Return 7

=head1 DESCRIPTION

Pegex::Forth is a Forth interpreter engine that uses Pegex to drive it.
