=encoding utf8

=head1 NAME

OpenAPI::Client::OpenAI::Path::videos-extensions - Documentation for the /videos/extensions path.

=head1 OPERATIONS

=head2 POST /videos/extensions

=head3 CreateVideoExtend

  $client->create_video_extend({
      body => { ... },
  });

Create an extension of a completed video.

=head4 Responses

B<200 - Success>

Content-Type: application/json

Example:

    {
       "completed_at" : 0,
       "created_at" : 0,
       "error" : {
          "code" : "string",
          "message" : "string"
       },
       "expires_at" : 0,
       "id" : "string",
       "model" : null,
       "object" : "video",
       "progress" : 0,
       "prompt" : "string",
       "remixed_from_video_id" : "string",
       "seconds" : "string",
       "size" : null,
       "status" : null
    }

=head1 SCHEMAS

=head2 CreateVideoExtendJsonBody

Properties:

=over

=item * C<prompt> (string, required) - Updated text prompt that directs the extension generation.

=item * C<seconds> (VideoSeconds, required) - Length of the newly generated extension segment in seconds (allowed values: 4,
8, 12, 16, 20).

See L</VideoSeconds> below for shape.

=item * C<video> (VideoReferenceInputParam, required) - Reference to the completed video to extend.

See L</VideoReferenceInputParam> below for shape.

=back

=head2 CreateVideoExtendMultipartBody

Properties:

=over

=item * C<prompt> (string, required) - Updated text prompt that directs the extension generation.

=item * C<seconds> (VideoSeconds, required) - Length of the newly generated extension segment in seconds (allowed values: 4,
8, 12, 16, 20).

See L</VideoSeconds> below for shape.

=item * C<video> (oneOf, required)

=back

=head2 VideoModel

See L<https://platform.openai.com/docs/api-reference> for details.

=head2 VideoReferenceInputParam

Properties:

=over

=item * C<id> (string, required) - The identifier of the completed video.

=back

=head2 VideoResource

Properties:

=over

=item * C<completed_at> (anyOf, required)

=item * C<created_at> (integer, required) - Unix timestamp (seconds) for when the job was created.

=item * C<error> (anyOf, required)

=item * C<expires_at> (anyOf, required)

=item * C<id> (string, required) - Unique identifier for the video job.

=item * C<model> (VideoModel, required) - The video generation model that produced the job.

See L</VideoModel> below for shape.

=item * C<object> (string, required) - The object type, which is always C<video> .

Allowed values: video

Default: video

=item * C<progress> (integer, required) - Approximate completion percentage for the generation task.

=item * C<prompt> (anyOf, required)

=item * C<remixed_from_video_id> (anyOf, required)

=item * C<seconds> (string, required) - Duration of the generated clip in seconds. For extensions, this is the
stitched total duration.

=item * C<size> (VideoSize, required) - The resolution of the generated video.

See L</VideoSize> below for shape.

=item * C<status> (VideoStatus, required) - Current lifecycle status of the video job.

See L</VideoStatus> below for shape.

=back
