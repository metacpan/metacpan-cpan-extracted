# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Business::UPS, please report it
responsibly.

**Contact:** Open a GitHub issue at
https://github.com/cpan-authors/Business-UPS/issues or email the current
maintainer via their CPAN address (see META.json or https://metacpan.org/author/TODDR).

Please include:

- A description of the vulnerability
- Steps to reproduce
- Any potential impact

We will acknowledge receipt within 7 days and aim to release a fix promptly.

## Supported Versions

Only the latest CPAN release is supported with security updates.
