=head1 NAME

XML::eXistDB::Util - Constants and other general functions

=head1 INHERITANCE

 XML::eXistDB::Util
   is a Exporter

=head1 SYNOPSIS

  use XML::eXistDB::Util;

L<XML::Compile::Util|XML::Compile::Util>.
