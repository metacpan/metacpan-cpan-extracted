# Contributing to Crypt-RIPEMD160

Thank you for considering contributing to Crypt-RIPEMD160.

## How to Contribute

1. Fork the repository on GitHub:
   https://github.com/cpan-authors/Crypt-RIPEMD160
2. Create a feature branch from `main`
3. Make your changes
4. Run the test suite:
   ```
   perl Makefile.PL
   make
   make test
   ```
5. Submit a pull request

## Reporting Bugs

Please open an issue on GitHub:
https://github.com/cpan-authors/Crypt-RIPEMD160/issues

## Coding Guidelines

- Follow existing code style and conventions
- Add tests for new functionality
- Update documentation as needed
- Keep commits focused — one logical change per commit

## Security Issues

For security vulnerabilities, please see [SECURITY.md](SECURITY.md) for
responsible disclosure instructions.
