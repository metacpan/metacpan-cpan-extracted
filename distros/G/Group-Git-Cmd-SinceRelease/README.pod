=head1 Group::Git::Cmd::SinceRelease

Provides the L<Group::Git> sub sommand since-release which gets the number
of commits each repository is ahead of the last release.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
