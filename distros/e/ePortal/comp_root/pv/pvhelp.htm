All rights reserved. %# This program is open source software %# %# %#---------------------------------------------------------------------------- Настройка домашней страницы
  • Персональная настройка "Домашней страницы" позволит вам изменить ее внешний вид так, чтобы наиболее важная и интересная для вас информация находилась на расстоянии "одного щелчка" мышью.
  • По умолчанию вы видите домашнюю страницу, которую настроил для вас администратор сервера. Вы не можете как-либо ее изменить.
  • Чтобы создать свою личную домашнюю страницу вам необходимо зарегистрироваться!
  • Настройка вашей персональной домашней страницы заключается в том, что вы можете по своему желанию добавлять или удалять с экрана различные информационные разделы.
  • Для добавления раздела надо внизу соответствующего столбца выбрать его название и нажать кнопку .
  • Для удаления любого раздела с экрана щелкните на картинке <% img( src => "/images/ePortal/dlg_x.png", alt => "Кнопка удаления раздела", href => "#", onClick => "javascript:alert('Да, да. Именно так!');" ) %>, расположенной в верхней правой части раздела.
  • Некоторые разделы рядом с кнопкой удаления имеют еще одну: <% img( src => "/images/ePortal/dlg_edit.png", alt => "Кнопка настройки раздела", href => "#", onClick => "javascript:alert('Да, да. Именно так!');" ) %>. Эта кнопка служит для настройки конкретного раздела. Например, вы можете настроить список меню или количество новостей в соответствующих разделах.
  • Кнопки <% img( src => "/images/ePortal/dlg_min.png", alt => "Кнопка свертывания раздела", href => "#", onClick => "javascript:alert('Да, да. Именно так!');" ) %> и <% img( src => "/images/ePortal/dlg_max.png", alt => "Кнопка развертывания раздела", href => "#", onClick => "javascript:alert('Да, да. Именно так!');" ) %> помогут вам в случае, если необходимо временно убрать с экрана нечасто используемый раздел.
