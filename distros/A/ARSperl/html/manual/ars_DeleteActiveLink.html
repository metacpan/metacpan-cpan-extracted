ars_DeleteActiveLink(ctrl, name)

Delete the named active link from the server connected to in the given control structure. Requires Administrator privs to use.

On success
Returns 1.
On failure
Returns 0.

Example: 

      ars_DeleteActiveLink($ctrl, "AL-FooBar") || die "$ars_errstr";
ars_DeleteActiveLink was introduced in version 1.5 of ARSperl
