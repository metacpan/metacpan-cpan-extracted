=head1 Locale::XGettext::TT2

Extract translatable strings from template files for the
Template Toolkit version 2.

=head1 SYNOPSIS

    xgettext-tt2 --help

=head1 DESCRIPTION

B<Locale::XGettext::TT2> is the parser backend for
L<xgettext-tt2>.  It is based on L<Locale::XGettext>.

See L<https://github.com/gflohr/Template-Plugin-Gettext>
for the big picture.

See L<Locale::XGettext> for an overview of all methods.

See L<xgettext-tt2> (or try "perldoc xgettext-tt2" or
"man xgettext-tt2") for information on invocation.

See L<Template::Plugin::Gettext> to learn more about how
to mark strings in templates as translatable.
