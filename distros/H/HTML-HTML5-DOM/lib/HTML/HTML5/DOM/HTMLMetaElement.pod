=head1 NAME

HTML::HTML5::DOM::HTMLMetaElement - implementation of the HTMLMetaElement interface of the HTML DOM

=head1 DESCRIPTION

HTML::HTML5::DOM::HTMLMetaElement is an implementation of the HTMLMetaElement interface of the HTML DOM. See L<HTML::HTML5::DOM> for a list of the conventions that have been used when translating the DOM to Perl.

=head2 HTML Elements

This class applies to the following HTML elements.

=over

=item * C<< {http://www.w3.org/1999/xhtml}meta >>

=back

=head2 Inheritance

HTML::HTML5::DOM::HTMLMetaElement inherits methods from the following Perl classes.

=over

=item * L<HTML::HTML5::DOM::HTMLElement>

=item * L<XML::LibXML::Augment::Element>

=item * L<XML::LibXML::Augment::Node>

=item * L<XML::LibXML::Element>

=item * L<XML::LibXML::Node>

=item * L<XML::LibXML::QuerySelector>

=item * L<HTML::HTML5::DOMutil::FancyISA>

=back

=head2 Additional Methods

As well as its inherited methods, this class provides the following methods.

=over

=item * C<< content >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("content") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< httpEquiv >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("http-equiv") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< name >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("name") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< scheme >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("scheme") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=back
