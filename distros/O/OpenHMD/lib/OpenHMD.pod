=head1 NAME

OpenHMD - Collection of modules for using OpenHMD

=head1 DESCRIPTION

OpenHMD is a distribution of modules for utilising OpenHMD within Perl.
It aims to provide both low level bindings and object-orientated wrappers.

=head1 SEE ALSO

=over 4

=item L<openhmd.net|http://openhmd.net/>

OpenHMD's website.

=item #openhmd @ freenode

Official IRC channel for OpenHMD

=item L<OpenHMD::Backend::Inline>

Thin binding module for OpenHMD utilising Inline::C
