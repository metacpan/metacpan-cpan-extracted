
# NAME

Protocol::OVSDB - Open vSwitch database protocol ([RFC 7047](https://www.rfc-editor.org/rfc/rfc7047.html))

# VERSION

1.0

# SYNOPSIS

```perl

use Protocol::OVSDB;

open my $fh, '+<:raw', '/run/openvswitch/db.lock'
  or die "Cannot open Open vSwitch socket: $!";
my $ovsdb = Protocol::OVSDB->new(
  on_send => sub($data) {
    print $fh $data;
    flush $fh;
  }
);

my $buf = '';
while (sysread($fh, $buf, 1024)) {
  $ovsdb->receive($buf);
}

# Note: the above is just the structure
# use Future::IO or IO::Async for real-world, non-blocking code

```

# DESCRIPTION

Enables interaction with Open vSwitch (OVS) and Open Virtual Network (OVN)
controller databases - or a custom database (ovsdb-server allows creation of
other databases than the OVS and OVN databases).
