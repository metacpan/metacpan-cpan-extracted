
# NAME

Task::WebDyne::Starman - Optional Starman integration for WebDyne

# SYNOPSIS

module synopsis
```bash
#  Using CPAN
#
cpan Task::WebDyne::Starman


#  Using CPANMinus
#
cpanm Task::WebDyne::Starman
```

# DESCRIPTION

Task::WebDyne::Starman install WebDyne, Plack and Starman modules
