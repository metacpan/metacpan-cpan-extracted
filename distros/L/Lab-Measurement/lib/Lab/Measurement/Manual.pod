=head1 NAME

Lab::Measurement::Manual - API documentation manual for Lab::Measurement

=head1 VERSION

version 3.810

=head1 OVERVIEW

Currently, the Lab::Measurement API consists of:

=over

=item *

L</Lab::Moose>, the recommended, current API, based on B<Modern Perl>
techniques

=item *

the historical API, consisting of the Lab::Instrument virtual
instruments and the high-level Lab::XPRESS interface

=back

=head1 Legacy Lab::Measurement API

This part is deprecated and will be removed with Lab::Measurement 3.9. 
Please base new measurements on the Moose-based API and port your old scripts.
For documentation, see the module files.
