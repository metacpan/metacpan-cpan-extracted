=head1 INSTALLATION

=head2 Installing to system perl

App::FromUnixtime installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan App::FromUnixtime

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install

=head2 Downloading the standalone executable

You can also copy the standalone executable to whatever location you'd like.

    % cd ~/bin
    % curl -LO http://bit.ly/from_unixtime
    % chmod +x from_unixtime

NOTE: edit shebang if you don't have /usr/bin/env
