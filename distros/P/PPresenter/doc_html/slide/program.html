 Portable Presenter is a package designed to give presentations. Presentations need some dynamics, to be attractive, although most people tend to use it too much.

A program is created when a slide has to be displayed. The information about the program is detected by the formatter. This is a string, for instance "phase 2 after 3". The string is passed on to the selected dynamic style-element, which translates this in a program-object: an internal structure of PPresenter.

When the slide is shown, PPresenter runs the program for you. This means that you have direct dealing with the program, but can contribute to it indirectly. End of discussion.
