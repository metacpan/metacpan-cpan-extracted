=head1 NAME

OpenAPI::Client::OpenAI::Path::evals-eval_id-runs-run_id-output_items-output_item_id - Documentation for the /evals/{eval_id}/runs/{run_id}/output_items/{output_item_id} path.

=head1 OPERATIONS

=head2 GET /evals/{eval_id}/runs/{run_id}/output_items/{output_item_id}

=head3 getEvalRunOutputItem

  $client->get_eval_run_output_item({
      body => { ... },
  });

Get an evaluation run output item by ID.

=head4 Path/query parameters

=over

=item * C<eval_id> (in path, required, string) - The ID of the evaluation to retrieve runs for.

=item * C<run_id> (in path, required, string) - The ID of the run to retrieve.

=item * C<output_item_id> (in path, required, string) - The ID of the output item to retrieve.

=back

=head2 EvalRunOutputItem

Properties:

=over

=item * C<created_at> (integer, required) - Unix timestamp (in seconds) when the evaluation run was created.
