=head1 NAME

Mylisp - my love Lisp

=head1 VERSION

Version 2.03

=head1 SYNOPSIS

Mylisp is a programming language like lisp.

    > mylisp --repl

then shell would ouput:

    This is Mylisp REPL. type 'exit' to exit.
    >>>
    
Lint mylisp.file:

    > mylisp --lint mylisp.file

transfer mylisp.file to Perl code.

    > mylisp --toperl mylisp.file

Mylisp also could transfer to Golang:

    > mylisp --togo mylisp.spp
    
Mylisp also could use as Module:

    use 5.012;
    use Mylisp qw(mylisp_to_ast get_mylisp_parser);

=head1 EXPORT

   mylisp_to_ast
   get_mylisp_parser
