    README for PDL::EditDistance

ABSTRACT
    PDL::EditDistance - Wagner-Fischer edit distance and alignment for PDLs.

REQUIREMENTS
    *   PDL

        Tested version(s) 2.4.2, 2.4.3, 2.4.7_001, 2.4.9_015.

DESCRIPTION
    PDL::EditDistance provides methods to compute Wagner-Fischer edit
    distance and alignment matrices for PDLs.

BUILDING
    Build this module as you would any perl module, by doing something akin
    to the following:

     gzip -dc distname-XX.YY.tar.gz | tar -xof -
     cd distname-XX.YY/
     perl Makefile.PL
     make
     make test                                     # optional
     make install

    See perlmodinstall(1) for details.
