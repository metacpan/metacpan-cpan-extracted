seekdir

ios::seekdir

Possible values are:

ios::beg 

  The step size relative to the beginning of the stream. (Default).

ios::cur 

  The step size relative to the current position (as returned by tellp() of the stream).

ios::end 

  The step size relative to the end position of the the stream.
