=head1 NAME

L<Novel::Robot::Parser>

get novel / bbs content from website

小说站点解析引擎


=head1 SUPPORT SITE


=head2 support download novel from website


支持小说/论坛站点下载

%SITE_DOM_NAME in Parser.pm 站点列表


=head2 support txt file

支持处理txt

txt : 指定解析txt文件


=head1 FUNCTION


=head2 new

init funtion, need set sitename，or url  初始化解析模块，需指定站点名称或网址

       #site name : 直接指定站点
       my $parser = Novel::Robot::Parser->new( site => 'jjwxc' );
        
       #url : 通过url自动检测站点
       my $url = 'http://www.jjwxc.net/onebook.php?novelid=2456';
       my $parser = Novel::Robot::Parser->new( site => $url );

       #site name : default，不明确指定站点名称
       my $url = 'http://www.zhonghuawuxia.com/book/71';
       my $class = 'novel'; # novel, tiezi, page
       my $parser = Novel::Robot::Parser->new( site => 'default', class => $class );




=head2 parse_novel

解析作者名，书名

    my $r = $parser->parse_novel($index_content_ref, %opt);


=head2 parse_item_list

解析章节页 

    $r = $parser->parse_item_list($index_content_ref, $r);

=head2 parse_novel_item

解析章节内容

    my $r = $parser->parse_novel_item($chapter_content_ref);

=cut
