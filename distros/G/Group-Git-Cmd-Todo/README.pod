=head1 Group::Git::Cmd::Todo

Adds the sub-command C<todo> to C<group-git>, it concatenates all the
C<TODO.md> files so you can view a summary of tasks to to accross all
repositories.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
