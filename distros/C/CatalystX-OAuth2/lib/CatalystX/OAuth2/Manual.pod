=head1 DESCRIPTION

L<CatalystX::OAuth2> is an oauth2 toolkit for L<Catalyst> implementing a
subset of the L<oauth 2 ietf spec
draft|http://tools.ietf.org/html/draft-ietf-oauth-v2-23>. The goal of this
document is to provide a fast introduction and familiarization with the tools
included.
