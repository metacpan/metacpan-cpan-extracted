=head1 NAME

OpenAPI::Client::OpenAI::Path::responses-response_id-cancel - Documentation for the /responses/{response_id}/cancel path.

=head1 OPERATIONS

=head2 POST /responses/{response_id}/cancel

=head3 cancelResponse

  $client->cancel_response({
      body => { ... },
  });

Cancels a model response with the given ID. Only responses created with the
C<background> parameter set to C<true> can be cancelled.
L<Learn more|https://platform.openai.com/docs/guides/background> .

=head4 Path/query parameters

=over

=item * C<response_id> (in path, required, string) - The ID of the response to cancel.

=back
