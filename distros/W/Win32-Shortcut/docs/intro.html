Win32::Shortcut

Perl Module to deal with Windows Shortcut
Version 0.03 (07 Apr 1997)
by Aldo Calpini (dada@perl.it)

Introduction

 This module implements the Win32 IShellLink Interface to allow management of shortcut files from Perl.

Version History

 0.03 (07 Apr 1997)
  • The PLL file now comes in 2 versions, one for Perl version 5.001 (build 110) and one for Perl version 5.003 (build 300 and higher, EXCEPT 304).
  • Added an installation program which will automatically copy the right files in the right place.
0.02 (21 Jan 1997)
  • Added methods matching properties to reduce typing overhead (eg. Alt-123 and 125...).
0.01 (15 Jan 1997)
  • First public release.
  • Added "Set" and "Resolve" and the properties "Hotkey", "IconLocation" and "IconNumber".
0.01a (10 Jan 1997)
  • First implementation of the IShellLink interface (wow, it works!); can "Load", "Save", and modify properties of shortcut files.

Installation Instructions
  1. Run the INSTALL.BAT program.

  2. Run the TEST.PL script to see if everything works

  3. Have fun.
