Geo-OGC-Service-WMTS version 0.03
================================

A web map tile service.

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

Geo::OGC::Service
