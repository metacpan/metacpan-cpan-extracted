=head1 NAME

OpenAPI::Client::OpenAI::Path::realtime-calls-call_id-reject - Documentation for the /realtime/calls/{call_id}/reject path.

=head1 OPERATIONS

=head2 POST /realtime/calls/{call_id}/reject

=head3 reject-realtime-call

  $client->reject_realtime_call({
      body => { ... },
  });

Decline an incoming SIP call by returning a SIP status code to the caller.

=head4 Path/query parameters

=over

=item * C<call_id> (in path, required, string) - The identifier for the call provided in the
L<C<realtime.call.incoming>|https://platform.openai.com/docs/api-reference/webhook-events/realtime/call/incoming>
webhook.

=back

=head4 Responses

B<200 - Call rejected successfully.>

=head1 SCHEMAS

=head2 RealtimeCallRejectRequest

Properties:

=over

=item * C<status_code> (integer) - SIP response code to send back to the caller. Defaults to C<603> (Decline)
when omitted.

=back
