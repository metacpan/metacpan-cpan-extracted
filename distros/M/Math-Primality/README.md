# Math::Primality

Advanced Primality Algorithms using Math::GMPz

[![Build Status](https://secure.travis-ci.org/leto/math--primality.png)](http://travis-ci.org/leto/math--primality)

# Installation

To install this module, run the following commands:

	perl Build.PL
	./Build
	./Build test
	./Build install

# Support
