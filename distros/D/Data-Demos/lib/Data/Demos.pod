=head1 SYNOPSIS

    data-demo list    - List open data sources
    data-demo import  - Import an open data source into a local db
    data-demo migrate <story>  - Apply a Migration Story

=head1 DESCRIPTION

L<Data::Demos> is set of tools for quickly creating demo databases and
applying migration stories to them.

It has the following concepts:

=over

=item C<data-demo>

The command line tool for interacting with Data::Demos.

=item Sources

Open data provider web sites

=item Datasets

Various collections of data around a topic

=item Database

Specific database engines to import datasets into:

=over

=item * PostgreSQL

=item * MySQL

=item * SQLite

=item * more

=back

=item Schemata

Definitions of columns, types and relations for a dataset

=item Stories

Common tasks for dataset migrations

=item Functions

Named migration code that can be applied to a dataset (does a migration)

=item Demos

Visual views of the data that can show off interesting things about a dataset

=back

=head1 DESIGN

This project is primarily/initially a way to see various Perl database tools,
and specifically L<Class::DBIx::DeploymentHandler> in action. From there
people can add stories and actions for more and more useful things needed in
dealing with dataset migration.
