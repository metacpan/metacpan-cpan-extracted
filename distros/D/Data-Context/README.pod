=head1 Data-Context

Data-Context provides a way for configuration files to be stored hireachly
in file system paths. It also allows for the content of those configuration
files to be changed dynamically at runtime.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
