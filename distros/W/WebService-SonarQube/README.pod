=head1 WebService::SonarQube

Provides a library to talk to SonarQube server instances using the SonarQube
web sservice API interface.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
