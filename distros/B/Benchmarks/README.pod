=head1 INSTALLATION

Benchmarks installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Benchmarks

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install


=head1 DOCUMENTATION

Benchmarks documentation is available as in POD. So you can do:

    % perldoc Benchmarks
