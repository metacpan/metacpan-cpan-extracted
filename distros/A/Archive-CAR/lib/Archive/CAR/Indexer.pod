=head1 NAME

Archive::CAR::Indexer - Index generator for CAR archives

=head1 SYNOPSIS

    use Archive::CAR::Indexer;

    my $indexer = Archive::CAR::Indexer->new();
    my $index_data = $indexer->generate_index($blocks);

=head1 DESCRIPTION

Archive::CAR::Indexer is an internal utility used to generate sorted multihash indexes for CAR v2 archives. It
currently supports index format 0x0400 (Multihash Index Sorted).

=head1 METHODS

=head2 C<generate_index($blocks)>

Generates a binary index for an arrayref of block records. Each record must contain a C<cid> object and an C<offset>.
