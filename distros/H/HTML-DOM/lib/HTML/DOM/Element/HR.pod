
=head1 NAME

HTML::DOM::Element::HR - A Perl class for representing 'hr' elements in an HTML DOM tree

=head1 VERSION

Version 0.054

=head1 SYNOPSIS

  use HTML::DOM;
  $doc = HTML::DOM->new;
  $elem = $doc->createElement('hr');

  $elem->align(1); # set attribute
  $elem->size;     # get attribute
  $elem->tagName;
  # etc

=head1 DESCRIPTION

This class implements 'hr' elements in HTML::DOM trees. It 
implements the HTMLHRElement DOM interface and inherits from 
L<HTML::DOM::Element> (q.v.).

=head1 METHODS

In addition to those inherited from HTML::DOM::Element and its 
superclasses, this class implements the following DOM methods:

=over 4

=item align

=item noShade

=item size

=item width

=back

Each of these returns the corresponding HTML attribute. If you pass an
argument, it will become the new value of the attribute, and the old value
will be returned.
