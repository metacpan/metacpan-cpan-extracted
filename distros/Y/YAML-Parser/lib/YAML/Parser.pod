=head1 NAME

YAML::Parser - Generated Reference Parser for YAML 1.2

=head1 VERSION

This document describes L<YAML::Parser> version B<0.0.4>.

=head1 SYNOPSIS

To get a set of event objects:

    my @events = YAML::Parser->new->parse($yaml_string)->events;

To get the yaml-test-suite event output string:

    my $receiver = PerlYamlReferenceParserTestReceiver->new;
    my $output = YAML::Parser->new(receiver => $receiver)
     ->parse($yaml_string)
     ->receiver
     ->output;

=head1 DESCRIPTION

YAML::Parser is the first 100% YAML 1.2 spec compliant parser for Perl. The
Perl code is generated directly from the YAML 1.2 specification.
