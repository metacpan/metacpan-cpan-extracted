=head1 INSTALLATION

B<Text::Chord::Piano> installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Text::Chord::Piano

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % sudo make install
