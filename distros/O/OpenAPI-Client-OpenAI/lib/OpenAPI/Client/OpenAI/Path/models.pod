=head1 NAME

OpenAPI::Client::OpenAI::Path::models - Documentation for the /models path.

=head1 OPERATIONS

=head2 GET /models

=head3 listModels

  $client->list_models({
      body => { ... },
  });

Lists the currently available models, and provides basic information about
each one such as the owner and availability.

=head4 Responses

B<200 - OK>

Content-Type: application/json

Example:

    {
       "data" : [
          "{\n  \"id\": \"VAR_chat_model_id\",\n  \"object\": \"model\",\n  \"created\": 1686935002,\n  \"owned_by\": \"openai\"\n}\n"
       ],
       "object" : "list"
    }

=head1 SCHEMAS

=head2 ListModelsResponse

Properties:

=over

=item * C<data> (array of Model, required)

=item * C<object> (string, required)

Allowed values: list

=back

=head2 Model

Properties:

=over

=item * C<created> (integer, required) - The Unix timestamp (in seconds) when the model was created.

=item * C<id> (string, required) - The model identifier, which can be referenced in the API endpoints.

=item * C<object> (string, required) - The object type, which is always "model".

Allowed values: model

=item * C<owned_by> (string, required) - The organization that owns the model.

=back
