=head1 NAME

OpenAPI::Client::OpenAI::Path::organization-groups-group_id-users-user_id - Documentation for the /organization/groups/{group_id}/users/{user_id} path.

=head1 OPERATIONS

=head2 DELETE /organization/groups/{group_id}/users/{user_id}

=head3 remove-group-user

  $client->remove_group_user({
      body => { ... },
  });

Removes a user from a group.

=head4 Path/query parameters

=over

=item * C<group_id> (in path, required, string) - The ID of the group to update.

=item * C<user_id> (in path, required, string) - The ID of the user to remove from the group.

=back

=head4 Responses

B<200 - User removed from the group successfully.>

Content-Type: application/json

Example:

    {
       "deleted" : true,
       "object" : "group.user.deleted"
    }

=head2 GET /organization/groups/{group_id}/users/{user_id}

=head3 retrieve-group-user

  $client->retrieve_group_user({
      body => { ... },
  });

Retrieves a user in a group.

=head4 Path/query parameters

=over

=item * C<group_id> (in path, required, string) - The ID of the group to inspect.

=item * C<user_id> (in path, required, string) - The ID of the user to retrieve from the group.

    }
