# corpus/

Hand-crafted PDF fixtures used by the round-trip test suite.

Each fixture is either:

- **writer-produced** — generated by our own phase 04/05 writer, used to
  smoke-test the writer output against external validators
  (`mutool info`, Preview.app) while the matching parser doesn't yet exist.
- **hand-crafted** — authored by hand to exercise a specific spec clause
  (e.g. a minimal xref stream, a deep /Pages tree, a single-page doc with a
  known TTF subset).
- **real-world** (under `corpus/real-world/`) — a small set of PDFs with
  permissive licensing collected to stress-test the parser. These should
  stay small (a few MB total).

No fixture in this directory carries proprietary or confidential content.

Fixtures are populated phase-by-phase; phase 01 leaves this directory
empty on purpose. See `../plan/phase-05-hello-world.md` onwards for what
gets written here.
