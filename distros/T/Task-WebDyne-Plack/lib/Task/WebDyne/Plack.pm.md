
# NAME

Task::WebDyne::Plack - Optional Plack integration for WebDyne

# SYNOPSIS

module synopsis
```bash
#  Using CPAN
#
cpan Task::WebDyne::Plack


#  Using CPANMinus
#
cpanm Task::WebDyne::Plack
```

# DESCRIPTION

Task::WebDyne::Plack install WebDyne and Plack (including Starman) modules
