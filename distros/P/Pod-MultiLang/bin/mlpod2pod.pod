=encoding utf-8

=head1 NAME

mlpod2pod - MultiLang-pod into just a normal pod


=head1 SYNOPSIS

 mlpod2pod [options] mlpodfile > podfile

=head1 DESCRIPTION

turn multilang pod into just a normal pod.


=head1 OPTIONS

in arguments, if it starts with '-', there are options.
after '--', there are file list.


=over

=item --outfile="filename"

output to F<filename>.


=item -o "filename"

alias for --outfile="filename".


=item --auto-out

output to *.pod file. filename is auto-generated by input 
filename.


=item --langs=""

select output languages written in mlpod document.
separated by ','.


=back

=head1 ENVIRONMENT VARIABLES

=over

=item MLPOD_LANGS

same as --langs=$MLPOD_LANGS.
using only --langs is ommitted.


=back
