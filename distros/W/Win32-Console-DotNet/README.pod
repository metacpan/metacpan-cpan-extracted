=head1 Win32::Console::DotNet

=head2 INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install

=head2 SUPPORT AND DOCUMENTATION

After installing, you can find documentation for this module with the
perldoc command.

	perldoc Win32-Console-DotNet

You can also look for information at:

	GitHub's issues tracker (please report bugs here!)
		https://github.com/brickpool/Win32-Console-DotNet/issues

	GitHub
		https://github.com/brickpool/Win32-Console-DotNet

	Search CPAN
		https://metacpan.org/dist/Win32-Console-DotNet
