# WebDyne Examples

All examples in this directory are taken from the WebDyne documentation at
<https://webdyne.org>.  You can interact with the examples by starting the
WebDyne PSGI server and pointing to the directory these files live in:

`webdyne.psgi /path/to/examples`

Connect to the WebDyne server at the appropriate location and you will be
able to browse the files, run them and view the source code.
