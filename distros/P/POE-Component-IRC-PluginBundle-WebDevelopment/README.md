# NAME

POE::Component::IRC::PluginBundle::WebDevelopment - a collection of plugins useful for Web Development IRC bots

# CONTENTS

This distribution provides the following
[POE::Component::IRC](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC) plugins:

- [POE::Component::IRC::Plugin::AntiSpamMailTo](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AAntiSpamMailTo) generate `mailto:` links that avoid dumb spam bots
- [POE::Component::IRC::Plugin::BrowserSupport](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ABrowserSupport) lookup browser support for CSS/HTML/JS
- [POE::Component::IRC::Plugin::ColorNamer](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AColorNamer) tells the name of the color by its hex code
- [POE::Component::IRC::Plugin::CSS::Minifier](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ACSS%3A%3AMinifier) "minify" CSS code
- [POE::Component::IRC::Plugin::CSS::PropertyInfo](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ACSS%3A%3APropertyInfo) lookup CSS property information
- [POE::Component::IRC::Plugin::CSS::SelectorTools](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ACSS%3A%3ASelectorTools) couple of CSS selector tools
- [POE::Component::IRC::Plugin::Google::PageRank](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AGoogle%3A%3APageRank) non-blocking access to Google's PageRank via IRC
- [POE::Component::IRC::Plugin::HTML::AttributeInfo](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AHTML%3A%3AAttributeInfo) HTML attribute info lookup
- [POE::Component::IRC::Plugin::HTML::ElementInfo](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AHTML%3A%3AElementInfo) lookup HTML element information
- [POE::Component::IRC::Plugin::JavaScript::Minifier](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AJavaScript%3A%3AMinifier) PoCo::IRC plugin to minify JavaScript code
- [POE::Component::IRC::Plugin::Syntax::Highlight::CSS](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ASyntax%3A%3AHighlight%3A%3ACSS) highlight CSS code from URIs
- [POE::Component::IRC::Plugin::Syntax::Highlight::HTML](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3ASyntax%3A%3AHighlight%3A%3AHTML) highlight HTML code from URIs
- [POE::Component::IRC::Plugin::Validator::CSS](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AValidator%3A%3ACSS) non-blocking CSS validator
- [POE::Component::IRC::Plugin::Validator::HTML](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AValidator%3A%3AHTML) non-blocking HTML validator
- [POE::Component::IRC::Plugin::WWW::Alexa::TrafficRank](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AWWW%3A%3AAlexa%3A%3ATrafficRank) get Alexa traffic rank for pages
- [POE::Component::IRC::Plugin::WWW::DoctypeGrabber](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AWWW%3A%3ADoctypeGrabber) plugin to display DOCTYPEs and relevant information from given pages
- [POE::Component::IRC::Plugin::WWW::GetPageTitle](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AWWW%3A%3AGetPageTitle) web page title fetching IRC plugin
- [POE::Component::IRC::Plugin::WWW::HTMLTagAttributeCounter](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AWWW%3A%3AHTMLTagAttributeCounter) html tag and attribute counter
- [POE::Component::IRC::Plugin::WWW::Lipsum](https://metacpan.org/pod/POE%3A%3AComponent%3A%3AIRC%3A%3APlugin%3A%3AWWW%3A%3ALipsum) plugin to generate Lorem Ipsum text
