=head1 NAME

Graphics::Toolkit::Color::Manual::Deprecation - deprecated API parts

=head1 SYNOPSIS

This L<manual|Graphics::Toolkit::Color::Manual> page lists which keywords 
will be removed with the next major version (3.0).

=head1 METHODS

=head2 apply

was renamed to L<tone_curve|Graphics::Toolkit::Color::Manual::Calculation/tone_curve>.

=head1 ARGUMENTS

=head2 amount

The argument C<amount> of the L<mix|Graphics::Toolkit::Color::Manual::Calculation/mix> 
method got renamed to L<by|Graphics::Toolkit::Color::Manual::Argument/by>.


=head2 select

The argument C<select> of the L<distance|Graphics::Toolkit::Color::Manual::Getter/distance> 
method got renamed to L<only|Graphics::Toolkit::Color::Manual::Argument/only>.
