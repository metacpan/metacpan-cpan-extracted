=head1 NAME

Unix::Statgrab::sg_page_stats - container for libstatgrab paging statistics

=head1 DESCRIPTION

C<Unix::Statgrab::sg_page_stats> is a Perl package name providing
accessors to bundled libstatgrab L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html> paging statistics.

=head1 METHODS

=head2 nentries

Returns the number of entries in the container.

=head2 systime

Returns the seconds since Epoch (0:00:00 on January, 1st 1970) of the time measure the paging statistics.

=head2 as_list

B<Deprecated> alias of L</fetchall_hash>

=head2 fetchall_hash

Returns all records from L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html>
container as list of hash containing each datum name as key and the datum as value.

=head2 fetchall_array

Returns all records from L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html>
container as list of arrays containing the values in following order:

  (pages_pagein pages_pageout systime)

in that order without leading headline.

=head2 fetchall_table

Returns all records from L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html>
container as list of arrays containing the values in following order:

  (pages_pagein pages_pageout systime)

in that order including leading headline.

=head2 fetchrow_arrayref($row = 0)

Returns one row as array containing values as above.

=head2 fetchall_arrayref

Returns all records from L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html>
container as array of arrays without header.

=head2 fetchrow_hashref($row = 0)

Returns one row as hash containing C<< { pages_pagein => $self.pages_pagein($row), pages_pageout => ... } >>.

=head2 fetchall_hashref

Returns all records from L<sg_page_stats|http://www.i-scream.org/libstatgrab/docs/sg_get_page_stats.3.html>
container as array of hashes.

=head2 colnames

Returns list of column names: C<< (pages_pagein pages_pageout systime) >>

=head2 pages_pagein

The number of pages swapped into memory.

=head2 pages_pageout

The number of pages swapped out of memory (to swap).

=head2 get_page_stats_diff

Provides the difference between the last measurement and the recent one.

  $recent->get_page_stats_diff($last);
